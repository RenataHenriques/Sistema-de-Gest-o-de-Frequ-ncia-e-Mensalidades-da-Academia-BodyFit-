# Projeto: Sistema de Gestão de Matrículas e Pagamentos da Academia "BodyFit"

## 1. Descrição do Mini-Mundo

O minimundo consiste na gestão de clientes de uma academia. O sistema foca em registrar o cadastro do **ALUNO** (Nome, CPF, contato), controlar o **contrato de MATRÍCULA** (data de início, status) e processar os **PAGAMENTOS** recorrentes, vinculando cada transação ao seu respectivo contrato.

## 2. Implementação SQL e Entregáveis

Este repositório contém o modelo físico (DDL) e os comandos de manipulação de dados (DML) para o projeto, garantindo a Terceira Forma Normal (3FN) e a integridade referencial.

* **`01_DDL_Criacao_Tabelas.sql`**: Comandos CREATE TABLE para criar a estrutura do banco.
* **`02_DML_Manipulacao_Dados.sql`**: Comandos INSERT, SELECT, UPDATE e DELETE.
